incremento 
a) pós-fixado

num = 10;
x = num++ -7

x = num - 7;
num = num + 1;

==> x=3 ; num = 11

b) pré-fixado
 num = 3;
x = ++num * 5;

num = num + 1;
x = num * 5;

==> num = 4 ; x = 20;
